# mh-dynaspat 项目

您好！欢迎来到我的科研项目仓库。

## 📝 项目简介
这是我为了申请 **Pengyu ZHU** 教授课题组所准备的研究项目。
该项目主要关注 **Dynamic Spatial (动态空间)** 相关的数据分析与建模。

## 📁 目录结构说明
* `data/`: 存放研究使用的原始数据或处理后的数据。
* `scripts/`: 存放用于分析的 Python 脚本代码。
* `docs/`: 存放研究方案、套磁信草稿以及相关参考文献。

## 🚀 如何使用
1. 克隆本项目到本地：
   `git clone https://github.com/mahai-png/mh_dynaspat.git`
2. 按照 `docs/` 中的说明运行相关分析脚本。
